# PIDEONWASH | Digital Production Studio

## Übersicht

PIDEONWASH ist ein digitales Produktionsstudio, das sich auf die Erstellung immersiver und fesselnder Inhalte konzentriert. Unser Hauptprojekt ist **EXIRE**, ein narratives Erlebnis, das in einem parallelen Welt-Szenario spielt, gefüllt mit Drama, Kühnheit und unerwarteten Wendungen.

## Funktionen

- **Interaktive Homepage**: Präsentiert das Projekt EXIRE und bietet einen Überblick über das Studio.

- **Team-Seite**: Stellt die kreativen Köpfe hinter PIDEONWASH und EXIRE vor.

- **Projekt EXIRE**: Eine dedizierte Sektion für unser Hauptprojekt, die es Benutzern ermöglicht, die Welt von EXIRE zu erkunden.

- **Responsive Design**: Optimiert für verschiedene Bildschirmgrößen und Geräte.

## Technologien

Das PIDEONWASH-Projekt ist eine Webanwendung, die mit folgenden Technologien entwickelt wurde:

- **HTML5**: Für die Struktur der Webseiten.

- **CSS3**: Für das Styling und Layout, einschließlich responsiver Anpassungen.

- **JavaScript**: Für interaktive Elemente und dynamische Inhalte.

## Erste Schritte

Um das Projekt lokal auszuführen, klonen Sie das Repository und öffnen Sie die `index.html`-Datei in Ihrem Webbrowser.

```bash
git clone [REPOSITORY_URL]
cd PIDEONWASH
open index.html # oder doppelklicken Sie auf die Datei
```
